# Tech Blog
[Deployed on Heroku!](https://random-badger.herokuapp.com/)
## Description
App built with Model-View-Controller (MVC) architecture where users can sign up and post to a blog style website. User info can be edited and saved as well as make comments on other user's posts.
- Javascript
- Node.js
- Express.js
- dotenv
- express-handlebars
- expression-session
- MySQL
- Sequelize
- Heroku
